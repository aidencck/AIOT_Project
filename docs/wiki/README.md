## 文档约定

- 状态标签：`current`（已落地）、`in-progress`（开发中）、`planned`（规划中）
- 事实来源：以仓库代码、`docker-compose.yml`、`pom.xml`、`ci-cd.yml` 为准
- 变更原则：修改功能代码时，同步更新对应 Wiki 页面
